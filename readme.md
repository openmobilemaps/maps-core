## Features
* Multi-platform graphics engine based on OpenGL for Android and Metal for iOS
* Touch and gesture handling
* Tiled Map Layers
* Vector Layers
* Polygon Layers
* Icon Layers

...and more to come! See our roadmap at [openmobilemaps.io](https://openmobilemaps.io)
